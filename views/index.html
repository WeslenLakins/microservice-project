WTL Timestamp Microservice

This is my project of the Timestamp Microservice challenge for the freeCodeCamp API and Microservice certification. It was built based on the boilerplate available here.

Example Usage:

To get the current timestamp:

api/timestamp

To get a timestamp from a specific date:

api/timestamp/2015-12-25

Or from a Unix timestamp:

/api/timestamp/1451001600000

Example Output:

{"unix":1451001600000, "utc":"Fri, 25 Dec 2015 00:00:00 GMT"}
